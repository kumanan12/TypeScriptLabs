Learn ES6 using TypeScript - let and var

Using var now

In ES5, variables are

  1. function scoped.
  2. hoisted to the to the top of the function.

var-and-let.ts

							

    var x = 3;
    function func(randomize) {
        if (randomize) {
            var x = Math.random();
            return x;
        }
        return x;
    }
    var result = func(false); // undefined
    toastr.info("result= " + result);
