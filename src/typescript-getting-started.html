Installing and configuring TypeScript

  • Install TypeScript

    						

	npm install -g typescript

    g - global

  • Configure TypeScript

    1. tsconfig.json - Options for the TypeScript compiler 
      					   
						   {
    "compilerOptions": {
        "target": "es5",
        "module": "commonjs",
        "sourceMap": true
    }
}
